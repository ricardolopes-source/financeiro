📋 Passo a passo — Supabase Gratuito

1
Criar conta gratuita

Acesse supabase.com → clique em "Start your project" → entre com sua conta GitHub ou crie um e-mail/senha.

2
Criar novo projeto

Clique em "New project" → dê o nome tempo-real → escolha uma senha forte → região South America (São Paulo) → clique em "Create new project". Aguarde ~2 minutos.

3
Criar as tabelas no banco

No menu lateral, clique em "SQL Editor" → cole e execute o SQL abaixo:

create table if not exists clientes (id text primary key, data jsonb not null, updated_at timestamptz default now()); create table if not exists areceber (id text primary key, data jsonb not null, updated_at timestamptz default now()); create table if not exists folha (id text primary key, data jsonb not null, updated_at timestamptz default now()); create table if not exists gastos (id text primary key, data jsonb not null, updated_at timestamptz default now()); create table if not exists gcli (id text primary key, data jsonb not null, updated_at timestamptz default now()); alter table clientes enable row level security; alter table areceber enable row level security; alter table folha enable row level security; alter table gastos enable row level security; alter table gcli enable row level security; create policy "acesso_total" on clientes for all using (true) with check (true); create policy "acesso_total" on areceber for all using (true) with check (true); create policy "acesso_total" on folha for all using (true) with check (true); create policy "acesso_total" on gastos for all using (true) with check (true); create policy "acesso_total" on gcli for all using (true) with check (true);
4
Pegar as credenciais

No menu lateral: "Project Settings" → "API". Copie a Project URL e a chave anon public. Projetos novos usam o formato sb_publishable_…; projetos antigos usam eyJ… — ambos funcionam.

5
Colar e conectar

Volte para esta tela, cole os dois valores nos campos e clique em "Conectar". Pronto — todos os dispositivos verão os mesmos dados.

💡 O plano gratuito do Supabase inclui 500 MB de banco, sem limite de requisições e sincronização entre dispositivos. Ideal para equipes pequenas.
